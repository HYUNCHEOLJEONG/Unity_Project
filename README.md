# 2D 슈팅 게임 - Dragon Flight

Unity로 만든 라운드 기반 2D 슈팅 게임입니다.  
적 스폰, 보스 등장, 라운드 시스템, 플레이어 UI 등을 포함합니다.

---

## 주요 기능

- 플레이어 이동 및 공격
- 무기 시스템
- 적 스폰 + 점점 어려워지는 라운드
- 10라운드마다 보스 등장
- 체력/마나 UI 연동
- 승리 메시지 (`YOU WIN!`)
- 게임 종료 및 재시작 시스템

---

##  사용 기술

- Unity 2021.3.x
- C#
- TextMeshPro
- Git & GitHub
